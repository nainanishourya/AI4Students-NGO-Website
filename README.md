## ğŸš€ Getting Started

### Prerequisites
No special prerequisites needed! This is a static website built with pure HTML, CSS, and JavaScript.

### Installation & Setup

1. **Clone or download** this repository to your local machine

2. **Open the website**:
   - Simply open `index.html` in any modern web browser
   - Or use a local server for better development experience

3. **Using a local server** (recommended for development):
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

4. **Access the website**:
   - Direct file: `file:///path/to/AI4Students/index.html`
   - Local server: `http://localhost:8000`

## ğŸ¨ Customization

### Colors
The website uses CSS custom properties (variables) for easy customization. Edit the `:root` section in `css/style.css`:

```css
:root {
    --primary-color: #2563eb;      /* Main blue */
    --secondary-color: #7c3aed;    /* Purple */
    --accent-color: #10b981;        /* Green */
    --dark-color: #1e293b;          /* Dark text */
    --light-color: #f8fafc;         /* Light background */
}
```

### Content
- Edit HTML files directly to update text content
- Update statistics in `index.html` (Impact section)
- Modify contact information in all pages (footer section)

### Images
- Add logo: Create an `images/` folder and add your logo
- Update the logo section in all HTML files
- Add illustrations to resource cards and activity sections
